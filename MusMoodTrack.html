Musical Mood Tracker

Javascript
React

HTML / CSS
Spotify API

Journaling and Mood Tracking has proven to be effective in helping people be more aware and proactive about their mental well being. Wouldn't it be convenient if you could track your mental health automatically based on your behaviour? Our choice of music often reflects the way we're feeling.

Musical Mood Tracker uses the Spotify API to get a users recently played songs, extracts audio features from these songs and feeds it into a Mage.AI categorization model to get the users mood. The app then updates a Calender to reflect the users mood for the day, and keeps track of the user's mood over a period of time for a better understanding of the user's mental health and emotions.

My Role
  • Lead a team of 4 to conceptualize, plan and create a mood journaling web application.
  • Engineered Python scripts for to source dataset of 1000 songs that was then utilized to train a Mage.AI Machine Learning Model that predicted song mood with an accuracy of 70%.
  • Constructed Python backend to source user listening history to predict mood and efficiently integrated it with website frontend using React and Flask.
